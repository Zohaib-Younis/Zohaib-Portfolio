📖 Overview
This portfolio website is designed to showcase my work, skills, and professional background as a BSIT graduate and Full Stack Developer. The site features a modern dark theme with gradient accents, smooth animations, and a fully responsive layout that works across all devices.

✨ Features
🎨 Modern Dark Theme - Clean, professional dark design with gradient accents

📱 Fully Responsive - Optimized for all screen sizes from mobile to desktop

⚡ Smooth Animations - Framer Motion powered animations throughout

📄 Resume Download - One-click CV download functionality

📧 Contact Form - Email integration with mailto functionality

🔄 Testimonial Carousel - Auto-rotating client testimonials

🛠️ Skills Showcase - Visual display of technical skills and expertise

📁 Project Gallery - Featured projects with live demo links

🎯 Type Animation - Dynamic typing effect in hero section

🌐 Social Links - Integrated social media and contact links

🛠️ Tech Stack
Frontend
React.js - UI library

React Router DOM - Navigation and routing

Framer Motion - Animations and transitions

Lucide React - Icon library

Styling
CSS3 - Custom styles with CSS variables

Responsive Design - Mobile-first approach

Build Tools
Vite - Build tool and development server

npm - Package manager

📦 Installation
Clone the repository

bash
git clone https://github.com/Zohaib-Younis/portfolio-website.git
cd zohaib-portfolio
Install dependencies

bash
npm install
Start the development server

bash
npm run dev
Build for production

bash
npm run build
Preview the build

bash
npm run preview
🚀 Usage
Development
bash
npm run dev
The site will be available at http://localhost:5173

Production Build
bash
npm run build
npm run preview
